# PDFXS

轻量级 Windows PDF 阅读器，基于 Electron + PDF.js 构建。

---

## 安装

下载 `PDFXS Setup x.x.x.exe`，运行安装程序，按提示完成安装即可。

---

## 功能

### 打开文件
| 方式 | 操作 |
|------|------|
| 工具栏 | 点击「打开」按钮 |
| 快捷键 | `Ctrl+O` |
| 拖拽 | 将 PDF 文件拖入窗口 |
| 最近文件 | 在空白标签页点击文件名 |

### 多标签页
- `Ctrl+T` 或点击工具栏「＋」开启新标签页
- 点击标签页标题切换，点击 × 关闭
- 每个标签页独立记录阅读进度

### 翻页 & 导航
| 操作 | 说明 |
|------|------|
| `←` / `→` 或 `PageUp` / `PageDown` | 上一页 / 下一页 |
| `Home` / `End` | 跳转到第一页 / 最后一页 |
| 页码输入框 | 直接输入页码后按 Enter 跳转 |
| 目录侧栏 | `Ctrl+B` 打开，点击条目跳转 |

### 缩放
| 操作 | 说明 |
|------|------|
| `Ctrl++` / `Ctrl+-` | 放大 / 缩小（每次 25%）|
| `Ctrl+0` 或「适应」按钮 | 自适应窗口宽度 |
| 缩放输入框 | 直接输入数字（如 `150`）后按 Enter |
| `Ctrl+滚轮` | 缩放页面 |

### 文字选择 & 高亮
1. 在 PDF 正文中拖动鼠标选中文字
2. 选择工具栏中的颜色（黄 / 绿 / 蓝）
3. 点击「高亮」按钮或按 `Ctrl+H`

**管理高亮：**
- **删除**：双击已高亮区域，或右键选择「删除高亮」
- **修改颜色**：重新选中同一段文字，选择新颜色后再次高亮

高亮数据自动保存，下次打开同一文件时自动恢复。

### 搜索
1. `Ctrl+F` 打开搜索栏
2. 输入关键词后按 Enter 或点击「搜索」
3. `↑` / `↓` 按钮在结果页面间跳转
4. `Esc` 或点击 ✕ 关闭搜索栏

### 签名
1. 点击工具栏「签名」按钮
2. 在弹出画板上用鼠标绘制签名
3. 点击「确认并点击放置」
4. 在 PDF 页面上点击放置位置
5. 拖动已放置的签名可调整位置
6. 右键签名可删除

### 表单填写
对包含表单字段的 PDF，可直接在输入框中填写内容，数据自动保存。

### 夜间模式
- 点击工具栏「夜间」按钮，或按 `Ctrl+D`
- 将页面背景反转为深色，保护视力
- 彩色图片色相自动矫正，显示效果接近原色
- 再次点击或按快捷键退出夜间模式

## 快捷键一览

| 快捷键 | 功能 |
|--------|------|
| `Ctrl+O` | 打开文件 |
| `Ctrl+T` | 新标签页 |
| `Ctrl+W` | 关闭标签页 |
| `Ctrl+B` | 打开/关闭目录侧栏 |
| `Ctrl+F` | 打开/关闭搜索栏 |
| `Ctrl+H` | 高亮选中文字 |
| `Ctrl+D` | 切换夜间模式 |
| `Ctrl++` | 放大 |
| `Ctrl+-` | 缩小 |
| `Ctrl+0` | 适应窗口 |
| `←` / `→` | 上一页 / 下一页 |
| `Home` / `End` | 第一页 / 最后一页 |
| `F12` | 打开开发者工具 |

## 数据存储

应用数据保存在系统用户目录下：

```
C:\Users\<用户名>\AppData\Roaming\pdfxs\
├── progress.json       # 阅读进度（页码、缩放）
├── recent.json         # 最近打开文件列表
└── annotations\        # 高亮、签名、表单数据（按文件 MD5 命名）
```

## 开发

```bash
# 安装依赖
npm install

# 启动开发模式
npm start

# 打包为 Windows 安装包
npm run dist
```

打包产物在 `dist/` 目录下：`PDFXS Setup x.x.x.exe`。

## 技术栈

- [Electron](https://www.electronjs.org/) — 桌面应用框架
- [PDF.js](https://mozilla.github.io/pdf.js/) — PDF 渲染引擎（Mozilla）
- [electron-builder](https://www.electron.build/) — 打包工具
