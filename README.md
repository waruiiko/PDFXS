# PDFXS

轻量级 Windows PDF / EPUB / Markdown 阅读器，基于 Electron + PDF.js + epub.js + marked 构建。

## 功能

### 打开文件
| 方式 | 操作 |
|------|------|
| 工具栏 | 点击「打开」按钮 |
| 快捷键 | `Ctrl+O` |
| 拖拽 | 将 PDF / EPUB / Markdown 文件拖入窗口 |
| 最近文件 | 在空白标签页点击文件名 |

支持 **PDF**（.pdf）、**EPUB**（.epub）和 **Markdown**（.md）三种格式。

### 多标签页
- `Ctrl+T` 或点击工具栏「＋」开启新标签页
- 点击标签页标题切换，点击 × 关闭
- 每个标签页独立记录阅读进度
- PDF、EPUB、Markdown 可在不同标签页同时打开

### 翻页 & 导航
| 操作 | 说明 |
|------|------|
| `←` / `→` 或 `PageUp` / `PageDown` | 上一页 / 下一页（PDF：翻页；EPUB：上一章 / 下一章）|
| `Home` / `End` | 跳转到第一页 / 最后一页 |
| 页码输入框 | 直接输入页码后按 Enter 跳转 |
| 目录侧栏 | `Ctrl+B` 打开，点击条目跳转 |

### 缩放
| 操作 | 说明 |
|------|------|
| `Ctrl++` / `Ctrl+-` | 放大 / 缩小（每次 25%）|
| `Ctrl+0` 或「适应」按钮 | PDF：自适应窗口宽度；EPUB / Markdown：重置字号为 100% |
| 缩放输入框 | 直接输入数字（如 `150`）后按 Enter |
| `Ctrl+滚轮` | 缩放（仅 PDF）|

> EPUB / Markdown 的缩放以字号百分比表示（50%–300%）。

### Markdown 阅读
- 打开 `.md` / `.markdown` 文件，自动渲染为带样式的 HTML 页面
- 目录侧栏显示文档标题层级（H1–H6），支持折叠，点击跳转
- 支持代码块、表格、引用、图片等常见语法
- 缩放控制字号，夜间模式适配

### EPUB 阅读
- 连续滚动布局，逐章渲染，阅读体验流畅
- 目录侧栏完整显示书籍章节结构，支持多级嵌套
- 章节间导航：工具栏翻页按钮 / 页码输入框 / 目录点击
- 搜索功能按章节扫描全文，定位匹配章节
- 支持夜间模式，自动反转背景与文字颜色

### 文字选择 & 高亮（PDF / EPUB）
1. 在正文中拖动鼠标选中文字
2. 选择工具栏中的颜色（黄 / 绿 / 蓝）
3. 点击「高亮」按钮或按 `Ctrl+H`

**管理高亮：**
- **删除**：右键点击已高亮区域 → 选择「删除高亮」（EPUB），或在笔记面板点击 ✕
- **修改颜色**：重新选中同一段文字，选择新颜色后再次高亮

高亮数据自动保存，下次打开同一文件时自动恢复。

### 笔记面板（仅 EPUB）
- `Ctrl+B` 打开侧栏，点击「笔记」标签查看所有高亮
- 点击笔记条目可直接跳转到原文位置
- 点击「导出 .md」将所有笔记按章节导出为 Markdown 文件

### 搜索
1. `Ctrl+F` 打开搜索栏
2. 输入关键词后按 Enter 或点击「搜索」
3. `↑` / `↓` 按钮在结果页面 / 章节间跳转
4. `Esc` 或点击 ✕ 关闭搜索栏

### 签名（仅 PDF）
1. 点击工具栏「签名」按钮
2. 在弹出画板上用鼠标绘制签名
3. 点击「确认并点击放置」
4. 在 PDF 页面上点击放置位置
5. 拖动已放置的签名可调整位置
6. 右键签名可删除

### 表单填写（仅 PDF）
对包含表单字段的 PDF，可直接在输入框中填写内容，数据自动保存。

### 阅读模式（点击翻页）
- 点击工具栏「阅读」按钮切换阅读模式
- 激活后，点击**左下角**翻到上一页，点击**右下角**翻到下一页
- **PDF**：瞬间跳页，无滚动动画，适合单页阅读
- **EPUB**：整屏切换，与分页模式一致
- 再次点击「阅读」退出，恢复普通模式
- 打开 EPUB 时自动进入阅读模式；打开 PDF 时自动退出

### 夜间模式
- 点击工具栏「夜间」按钮，或按 `Ctrl+D`
- **PDF**：将页面背景反转为深色，彩色图片色相自动矫正
- **EPUB**：切换深色主题（背景 / 文字颜色），注入到章节 iframe 中
- **Markdown**：切换深色背景配色
- 再次点击或按快捷键退出夜间模式

## 数据存储

应用数据保存在系统用户目录下：

```
C:\Users\<用户名>\AppData\Roaming\pdfxs\
├── progress.json       # 阅读进度（页码 / 章节、缩放）
├── recent.json         # 最近打开文件列表
├── annotations\        # 高亮、签名、表单数据（按文件 MD5 命名）
└── window-state.json   # 窗口大小和位置
```

## 开发

```bash
# 安装依赖
npm install

# 启动开发模式
npm start

# 打包为 Windows 安装包
npm run dist
```

打包产物在 `dist/` 目录下：`PDFXS Setup 1.3.2.exe`。

## 技术栈

- [Electron](https://www.electronjs.org/) — 桌面应用框架
- [PDF.js](https://mozilla.github.io/pdf.js/) — PDF 渲染引擎（Mozilla）
- [epub.js](https://github.com/futurepress/epub.js/) — EPUB 渲染引擎
- [marked](https://marked.js.org/) — Markdown 渲染引擎
- [electron-builder](https://www.electron.build/) — 打包工具
